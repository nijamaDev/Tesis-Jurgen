\section{Demonstrate the example and examples environments}

The example and examples environment
are defined
in the covington
\cite{covington2021}
package.

Demonstrate the example environment:
\begin{example}
  This is an example.
  This is an example.
\end{example}

Demonstrate the examples environment:
\begin{examples}
  \item First example.
  \item Second example.
\end{examples}
